# Notre Doc
This project is here to make a new website for a doctor.
He needs a platform which is secured, modern and fast.

For the moment it uses these:
+ Typescript
+ Svelte + SvelteKit
+ Svelte Material UI (SMUI)
+ Sass
+ Git
+ Zod
+ Vitest
+ Playwright

And SST, with:
+ PostGres
+ graphql
+ genql
+ Pothos
